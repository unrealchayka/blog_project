pip install -r requarements.txt

1)Для использования сообщений в телеграмм создайте бота в https://t.me/BotFather
потом скопируйте токен и вставте в config.settings.TELEGRAM_TOKEN
2)Для получения айди группы/чата https://t.me/getmyid_bot
3)Для использования openweathermap(главная страница), зарегистрируйтесь на сайте https://openweathermap.org
и полчуите api_key
